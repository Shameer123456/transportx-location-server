Android — How to install
1
Download the APK
Tap "Download Android APK" above. Your browser will download a .zip file containing the APK.
2
Extract the APK
Open your Downloads folder, extract the zip and find the .apk file inside.
3
Allow unknown sources
Go to Settings → Security (or Apps → Special Access) and enable "Install unknown apps" for your file manager or browser.
4
Tap to install
Tap the .apk file in your Downloads folder and follow the install prompts. Done — open TransportX and go online.
⚠️
These are development builds. The Android APK is a debug build and the iOS IPA is unsigned. They are intended for testing and internal use only. To update the app files, replace the files in the public/downloads/files/ folder on the server and redeploy.
